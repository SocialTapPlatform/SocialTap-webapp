SocialTap API Documentation

GET /api/users

Returns a list of all users (excluding the current user).

GET /api/chats

Returns all chat rooms the current user is a participant in.

GET /messages?chat_id=<int:chat_id>

Returns all messages in a given chat. Please make sure you include badge support.

Body:

{
        "badges": [Testbadge],
    "content": "Testmessage",
    "id": 18237,
    "timestamp": "Not important nor accurate. When it does start getting accurate, we will put a banner on this API page.",
    "username": "AppleTest"}
POST /send

Sends a message.

Body:

{
  "chat_id": 1,
  "content": "Hello!"
}
POST /api/chats/create

Creates a chat (DM or group chat).

Body:

{
  "user_ids": [2, 3],
  "name": "Group Name" // required for group chats
}
DELETE /api/chats/delete/<int:chat_id>

Deletes a chat (if the user is authorized).

POST /api/user/offline

Marks the current user as offline. Required when the client is closing.

POST /api/login

Authenticates a user with email and password. Accepts JSON or form-encoded data.

{
  "email": "user@example.com",
  "password": "your_password"
}

Success Response:

{
  "success": true,
  "username": "yourUsername",
  "user_id": 1
}

Error Response (Invalid credentials):

{
  "error": "Invalid credentials"
}
POST /api/logout

Logs out the current authenticated user and clears their session. It also calls the same thing /api/user/offline does.

Success Response:

{
  "success": true,
  "message": "Logged out successfully"
}

Error Response:

{
  "error": "Logout failed"
}
