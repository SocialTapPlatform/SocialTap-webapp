Authentication Required: All endpoints require user authentication. Ensure you're logged in to use the API.

1. Send Message (Currently not working)

URL: /send

Method: POST

Authentication: Required (User must be logged in)

Request Body (Form Data)

  • message (string): The content of the message being sent.
  • chat_id (integer, optional): The ID of the chat room to send the message to.

Response (Success)

{
  "success": true,
  "message": {
    "id": 123,
    "content": "Hello, world!",
    "username": "user1",
    "timestamp": "14:30"
  }
}

Response (Error)

{
  "error": "Message cannot be empty"
}

2. Delete Chat

URL: /api/chats/delete/{chat_id}

Method: DELETE

Authentication: Required (User must be logged in)

URL Parameters

  • chat_id (integer): The ID of the chat room to delete.

Request Body (JSON)

  • user_id (integer): The ID of the user requesting the deletion.

Response (Success)

{
  "success": "Chat and all its messages have been deleted."
}

Response (Error)

{
  "error": "You do not have permission to delete this chat."
}

3. Get Chat Messages

URL: /messages?chat_id=0

Method: GET

Authentication: Required (User must be logged in)

URL Parameters

  • chat_id (integer): The ID of the chat room to retrieve messages from.

Response (Success)

{
  "messages": [
    {
      "id": 123,
      "content": "Hello, world!",
      "username": "user1",
      "timestamp": "14:30"
    },
    {
      "id": 124,
      "content": "Hi!",
      "username": "user2",
      "timestamp": "14:32"
    }
  ]
}

Response (Error)

{
  "error": "Chat room not found."
}

Error Codes

  • 400 - Bad Request (e.g., missing or invalid parameters)
  • 403 - Forbidden (e.g., user doesn't have permission to access a resource)
  • 404 - Not Found (e.g., the requested chat room doesn't exist)
  • 500 - Internal Server Error (e.g., unexpected error)
